# CSS

- short for `Cascading Style Sheets`

## Basic CSS

```css
/* comment */
/*
selector {
  property: value;
}
*/
<style>
  /* style element */
  h1 {
    /* selector */
    color: blue;
    /* property: value */
    text-align: center;
    /* property: value */
  }
</style>
```

## CSS Properties

- color: text color
- background-color: background color
- font-size: text size
- font-family: text font
- text-align: text alignment
- text-decoration: text decoration
- padding: padding
- margin: space
- border: border
- width: width
- height: height
